# NativeBase Hackathon

## Usage

### Command for Expo Project creation with NativeBase template

expo init my-app --template @native-base/expo-template

## Technologies Used:-

1) Expo
2) React-Native
3) Native Base
4) React Navigation
5) React Native Snap Carousel
6) React Native Youtube Iframe
7) Firebase
